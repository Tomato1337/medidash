# AI Service

## Назначение

AI Service предоставляет все операции, связанные с искусственным интеллектом:

- **Анонимизация текста** (PII removal) - удаление персональных данных
- **Векторизация текста** - создание embeddings для семантического поиска
- **Автоматическое тегирование** документов
- **Извлечение даты** из документа
- **Генерация названия** документа
- **Суммаризация** (summarization) нескольких документов в один

## Технологии

- NestJS
- OpenAI API (GPT-4, text-embedding-ada-002)
- Prisma ORM
- NLP библиотеки

## Основные операции

### Анонимизация (PII Removal)
- Обнаружение и замена:
  - ФИО
  - Адреса
  - Номера телефонов
  - Email адреса
  - Даты рождения
  - Медицинские ID
  - СНИЛС, паспортные данные
- Сохранение PII маппинга для возможной деанонимизации

### Векторизация
- Создание embeddings через OpenAI (1536 dimensions)
- Сохранение в PostgreSQL с pgvector

### Автоматическая метаинформация
- **Теги**: анализ содержимого и проставление релевантных тегов
- **Дата**: извлечение даты документа (или текущая дата)
- **Название**: генерация краткого описательного названия

### Суммаризация
- Объединение нескольких документов Record в один summary
- Сохранение ключевой информации

## Порт

По умолчанию: **3003**

## Переменные окружения

```env
AI_SERVICE_PORT=3003
OPENAI_API_KEY=sk-...
DATABASE_URL=postgresql://...
VECTOR_DIMENSIONS=1536
```

## Модели OpenAI

- **GPT-4**: анонимизация, тегирование, суммаризация
- **text-embedding-ada-002**: векторизация

## Запуск

```bash
pnpm install
pnpm start:dev
```

## API Endpoints

- `POST /ai/anonymize` - Анонимизация текста
- `POST /ai/vectorize` - Создание embeddings
- `POST /ai/auto-tag` - Автоматическое тегирование
- `POST /ai/summarize` - Суммаризация документов
